This code implements a complete, high-performance deep learning pipeline for detecting real vs. fake faces using PyTorch. It begins by importing necessary libraries, setting random seeds for reproducibility, and defining optimized data preprocessing techniques with balanced image augmentation to improve generalization without slowing training. The dataset is loaded using ImageFolder, and a stratified train-validation split ensures class balance. Efficient data loaders are configured with a larger batch size to speed up training.
The model architecture is based on a pretrained ResNet50 backbone, chosen for its strong balance between accuracy and speed. The original fully connected layer is replaced with a custom, deeper classifier that includes multiple linear layers, batch normalization, ReLU activations, and dropout for regularization. The training strategy is divided into two phases: first, the backbone is frozen and only the classifier is trained with a high learning rate for fast convergence; then, the entire network is unfrozen and fine-tuned using differential learning rates for better performance. A custom label smoothing loss function is used to reduce overconfidence, and advanced optimization techniques like AdamW, OneCycleLR, and cosine annealing are applied to accelerate and stabilize learning.
The training loop tracks loss, accuracy, and learning rate across epochs, with early stopping to prevent overfitting and automatic saving of the best-performing model. After training, the model is evaluated on the validation set, producing accuracy metrics and a detailed classification report. The code also generates visualizations including training curves and a confusion matrix, and saves all results (plots, model weights, and a summary report) to disk. Overall, this script is designed to efficiently achieve high accuracy (targeting 95%) within 25 epochs using a well-structured and optimized deep learning workflow.
